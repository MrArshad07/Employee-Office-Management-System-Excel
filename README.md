# Employee Office Management System (Microsoft Excel)

## Project Overview

The Employee Office Management System is a Microsoft Excel-based solution developed to manage employee information, attendance, leave records, company assets, employee documents, and business reporting.

The project simulates a real office administration system where employee records are maintained accurately using linked worksheets and automated Excel features.

---

## Project Objective

The objective of this project is to demonstrate practical skills in:

* Data Entry
* Office Administration
* Employee Record Management
* Document Management
* Attendance Tracking
* Leave Management
* Asset Tracking
* Excel Reporting
* Dashboard Creation

---

## Workbook Structure

The workbook contains six linked worksheets.

### 1. Employees

Stores employee master information.

Fields include:

* Employee ID
* First Name
* Last Name
* Department
* Designation
* Phone
* Email
* Joining Date
* Employment Status

---

### 2. Attendance

Tracks employee attendance.

Fields include:

* Attendance ID
* Employee ID
* Date
* Check-In
* Check-Out
* Working Hours
* Attendance Status

---

### 3. Leave

Maintains employee leave requests.

Fields include:

* Leave ID
* Employee ID
* Leave Type
* From Date
* To Date
* Number of Days
* Approval Status

---

### 4. Assets

Tracks company assets assigned to employees.

Fields include:

* Asset ID
* Employee ID
* Asset Name
* Serial Number
* Issue Date
* Return Date
* Asset Status

---

### 5. Documents

Maintains employee document verification.

Fields include:

* Employee ID
* Aadhaar
* PAN
* Resume
* Offer Letter
* Joining Form
* Verification Status

---

### 6. Dashboard

Provides management reports using Pivot Tables and Pivot Charts.

Key Performance Indicators include:

* Total Employees
* Active Employees
* Present Employees
* Absent Employees
* Approved Leave Requests
* Pending Leave Requests
* Assigned Assets
* Pending Document Verification
* Average Working Hours

---

## Dataset

* 500 Employee Records
* Linked using Employee_ID
* Separate tables for Attendance, Leave, Assets, Documents, and Dashboard

## Business Workflow

Employee information is stored once in the Employees sheet.

The Employee_ID is used as the common key to connect:

* Attendance
* Leave
* Assets
* Documents

This minimizes duplicate data and improves record accuracy.
